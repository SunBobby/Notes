# Git
> git操作汇总及备忘录
## 基础设置
- 初始化git仓库
```bash
git init
```
- 设置git账户
```bash
git config --global user.name "zcy"
git config --global user.email a@b.com
```
- 配置信息相关操作
```bash
git config --list #查看当前所有config信息
```
